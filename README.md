## Git Commit Rule
`Use Gitmoji`

 * 🎉 `:tada:` Begin a project
 * 📝 `:memo:` Writing docs
 * 🎨 `:art:` Improving the format/structure of the code
 * ✨ `:sparkles:` Introduce new features
 * 🐛 `:bug:` Fixing a bug
 * 🔧 `:wrench:` Add or update configuration files.
 * 🔥 `:fire:` Removing code or files
 * ♻️ `:recycle:` Refactor code
